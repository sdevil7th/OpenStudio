Studio13 Custom JSFX Effects
============================

Place your custom .jsfx scripts in this directory.
They will be bundled with Studio13 and appear in the Plugin Browser under "S13FX".

For user-created scripts that persist across updates, use:
  Documents/Studio13/Effects/

You can open the user effects folder from the Plugin Browser
by clicking the folder icon next to the Scan button.

JSFX documentation: https://www.reaper.fm/sdk/js/js.php
