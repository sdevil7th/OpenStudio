## Actual Visual Findings

- REAPER MIDI editor is dense and tool-light: a piano keyboard at left, a dark note grid, toolbar/ruler at top, track list at right, transport/CC controls at the lower-left edge, and one or more CC lanes below the note grid. Notes are strongly colored, velocity bars sit directly under corresponding notes, and CC-lane controls are compact rather than panel-heavy.
- REAPER CC lane behavior includes lane menus for existing lanes, explicit Off Velocity and Pitch lanes, scrollable lane selector, custom pitchwheel range, and per-lane vertical/horizontal zoom gestures.
- Cubase Key Editor is a full editor shell, not just a piano roll canvas: toolbar, status line, info line, visibility area, left-zone inspector, ruler/global tracks/track display, piano keyboard display, note display, and controller display.
- Cubase controller lanes show a dedicated info/edit strip above the grid, lane headers on the left, add/setup controls, a divider between note display and controller display, and curve/ramp rendering inside the lane.
- openDAW implements the note editor as separated components: `NoteEditor` composes header, piano roll, pitch canvas, property header, and property editor; `PitchEventCapturing` returns explicit capture targets for note body, note end, and loop duration; property editing is a separate canvas with draw/line/node modifiers; the property table exposes position, duration, pitch, velocity, cents, chance, play count, and play curve. That is the code-level parity target for Studio13, without copying AGPL code.

## First Recovery Acceptance Slice

The first implementation slice is considered useful only when this scenario passes in the real UI:

1. Create or load a MIDI clip on a MIDI/instrument track.
2. Drag it later in time and undo/redo.
3. Resize left and right edges and undo/redo.
4. Extend right edge beyond source length and verify loop notches/playback.
5. Alt-slip source content and undo/redo.
6. Ctrl/Cmd-drag to duplicate and undo/redo.
7. Split the MIDI clip and open the right split in the piano roll.
8. Use the piano roll select/draw/erase/resize tools and velocity lane without prompt dialogs.
